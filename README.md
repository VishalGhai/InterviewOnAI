# InterviewOnAI

A minimalist AI-powered interview preparation tool using Google Gemini.

## Setup

1. Get a Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey)
2. Open `config.js` and replace `YOUR_GEMINI_API_KEY_HERE` with your key
3. Open `index.html` in a browser (or use a local server)

## Usage

- **Job Description Mode**: Paste a job description, configure difficulty, and start
- **Topic Mode**: Click any technology button to start a focused interview

## Features

- 5-parameter answer evaluation (Accuracy, Depth, Clarity, Relevance, Practical Knowledge)
- Configurable question count and difficulty progression
- Real-time performance metrics sidebar
- Motivation milestone popups at 25%, 50%, 75%, 100%
- Final report card with strengths, weaknesses, and recommendations
